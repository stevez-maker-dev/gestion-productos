# Gestión de Productos — Angular

## Descripción
Aplicación desarrollada con Angular que permite gestionar una lista de productos.
Implementa servicios con inyección de dependencias, ciclo de vida ngOnInit,
pipes estándar (currency, date) y un pipe personalizado (descuento).

---

## Tecnologías utilizadas
- Angular 19
- TypeScript
- CSS

---

## Instrucciones para ejecutar el proyecto

### 1. Clonar el repositorio
git clone https://github.com/stevez-maker-dev/gestion-productos.git

### 2. Instalar dependencias
npm install

### 3. Ejecutar el proyecto
ng serve

### 4. Abrir en el navegador
http://localhost:4200

---

## Ejemplo de salida — Pipe personalizado descuento
Precio original: $85.000
Descuento aplicado: 15%
Precio final: $72.250
